测试内核实时性

安装测试脚本#

测试脚本在rm_bringup中，点击此处, 将此仓库拉到本地即可。

安装脚本依赖#

sudo apt install rt-tests stress gnuplot

进行测试#

进入脚本所在文件夹并运行实时性测试脚本#

cd rm_bringup/scriptssudo sh ./rh-test.sh

等待测试完成,测试结果会生成图片放在测试脚本所在文件夹#

以下图片为安装实时补丁前后的对比，可以明显看出安装补丁后实时性有了极大的提升。
