快速测试#

测试can#

  1. 初始化can0和can1
sudo ip link set can0 up type can bitrate 1000000sudo ip link set can1 up type can bitrate 1000000
  1. 监听can上的数据
sudo apt-get install can-utilscandump can0
tip

此时转动电机可监听到电机参数变化

  1. 发送数据驱动电机
cansend can0 200#05000000000000000

具体参见官方文档

修改串口物理地址#

  1. pc连接usbtoxxx.

  2. 查询usbImu和usbDbus物理地址

ls /sys/class/tty/ttyUSB* -l

得到如下：

其中，'3-2.3:1.0'为usbImu的物理地址

  1. 修改usbImu和usbDbus物理地址
cd /home/chenzheng/RM-Software/rm_ws/src/rm_bringup/scripts/udev/vim rm.rules

修改KERNELS=="${usb物理地址}"

./create_udev_rules.sh
  1. 查看是否映射成功
ls /dev/ | grep usb

存在usbDbus和usbImu即映射成功

tip

若找不到，重启；还是没有，换usb口重新执行上述步骤
